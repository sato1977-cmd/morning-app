# 朝の5分チャレンジ — 光里の朝ドリル

光里ちゃん（小6）専用の毎朝5分・iPad自習アプリ。
くもん「小学5年の総復習ドリル」をベースに、お小遣いシステムでモチベーションを保つ仕組み。

🌐 **公開URL**: https://sato1977-cmd.github.io/morning-app/

---

## 主な機能

- ⏱ **5分タイマー** — 毎日10問のミックス（算数・国語・理科・社会）または漢字書き取り
- 🐻 **くまさんヒント** — 段階的に詳しくなる2段階ヒント（使うと点数3分の1に）
- 💰 **お小遣いシステム** — 1点=1円、月間上限1,000円、連続日数ボーナス付き
- 🐷 **貯金箱** — 受け取り前の点数を蓄積、ボタンで現金化
- ⏸ **中断・🏠 リセット** — 5分タイマー中に一時停止／途中でやめる確認ダイアログ
- ✏️ **筆算スペース** — Apple Pencil対応の手書きキャンバス
- 📊 **履歴管理** — 全セッション・問題ごとの所要時間・ヒント使用を自動記録
- 📋 **CSVエクスポート** — エクセルで分析できる形式でクリップボードへ

---

## 使い方

### 光里ちゃん（毎朝）

1. iPadのホーム画面で 🐷 アイコンをタップ
2. ミックス／漢字を選んでスタート
3. 5分以内に10問やる
4. 結果画面で点数確認 → スタート画面で「受け取る」ボタンで貯金

### パパ・ママ（週1〜月1回）

1. iPadで「📊 履歴を見る」をタップ
2. 累計・受取済み・未受取の金額を確認
3. 「📋 履歴をコピー」or「📋 詳細をコピー」を押す
4. Macで `光里の朝ドリル_お小遣い管理.xlsx` を開きシートにペースト
5. 月間サマリ・分析タブで光里ちゃんの取り組み具合をチェック
6. 受取金額分の現金を渡す（光里ちゃんが「受け取る」を押した時点で履歴は受取済みになる）

## カスタマイズ

### 問題を追加・修正する

1. https://github.com/sato1977-cmd/morning-app/blob/main/index.html を開く
2. 鉛筆🖉アイコンで編集モード
3. `MIX_BANK` または `KANJI_BANK` 配列に追加
4. 「Commit changes」で確定 → 1分後に反映

### お小遣いレートを変える

`index.html` 内の以下の定数を編集：
- `YEN_PER_POINT` — 1点あたりの円
- `MONTHLY_LIMIT_YEN` — 月間上限
- `BONUS_PERFECT` / `BONUS_NO_HINT_ALL` — 全問正解・全問ノーヒントのボーナス
- `STREAK_BONUSES` — 連続日数ボーナス表

## 技術構成

- 📄 **単一ファイル** — `index.html` 1つで完結（HTML+CSS+JS全部入り）
- 🚫 **依存ゼロ** — 外部ライブラリ・CDN・APIなし、純粋なバニラJS
- 💾 **データ保存** — `localStorage`（キー: `hikari_morning_5min_v2`）。iPadのSafari内のみ
- 🎨 **キャンバス描画** — Apple Pencil + 指タッチ両対応、DPR対応
- 🌐 **配信** — GitHub Pages（無料・無期限）

---

## プライバシー

- このリポジトリは公開ですが、アプリ内のデータはiPadの `localStorage` 内だけ
- どこにも送信されません
- `index.html` 内に「光里」という名前と挨拶が含まれます
